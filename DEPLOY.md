# คู่มือการนำระบบขึ้นออนไลน์ฟรี (Render.com + Turso Database)

คู่มือนี้จะอธิบายขั้นตอนการอัปโหลดโค้ดระบบติดตามโครงการขึ้นออนไลน์แบบ **ฟรี 100%** โดยฝากโค้ดหลังบ้านไว้บน **Render.com (Free Plan)** และใช้บริการฐานข้อมูลแบบ Cloud SQLite บน **Turso Database (Free Plan)** ซึ่งข้อมูลโครงการทั้งหมดจะไม่สูญหายเมื่อระบบรีสตาร์ทตัวเอองครับ

---

## 🛠️ สิ่งที่ต้องเตรียมก่อนเริ่ม
1. บัญชี **GitHub** ([สมัครฟรีที่ github.com](https://github.com))
2. บัญชี **Render.com** ([สมัครฟรีที่ render.com](https://render.com))
3. บัญชี **Turso** ([สมัครฟรีที่ turso.tech](https://turso.tech)) *สามารถล็อกอินผ่านบัญชี GitHub ได้โดยตรง*

---

## ⚙️ ข้อมูลที่ต้องระวัง (Trade-off ของแผนบริการฟรี)
* **ฐานข้อมูล (Database)**: ปลอดภัยและเก็บถาวร 100% บน Turso Database (ฟรีสูงสุด 9GB)
* **ไฟล์แนบ / รูปโปรไฟล์ (Uploaded Files)**: เนื่องจาก Render.com แผนบริการฟรี (Free Plan) ไม่มีพื้นที่เก็บดิสก์ถาวร ไฟล์รูปโปรไฟล์ของทีมงานและไฟล์ PDF ใบเสร็จ/เอกสารอนุมัติงบประมาณที่อัปโหลดขึ้นไป จะถูกรีเซ็ตล้างข้อมูลทุกครั้งที่เซิร์ฟเวอร์หลับหรือรีสตาร์ทตัวเอง (แต่ข้อมูลตัวอักษร, แฟ้มโครงการ, ยอดเงินงบประมาณ, และแผนงาน Gantt จะคงอยู่ถาวร 100% บน Turso คลาวด์)

## 📋 ขั้นตอนการ Deploy ระบบ

### ขั้นตอนที่ 1: สร้างฐานข้อมูลบน Turso
1. ล็อกอินเข้าใช้งานเว็บ [Turso Dashboard](https://turso.tech)
2. คลิกปุ่ม **Create Database**
3. ตั้งชื่อฐานข้อมูล เช่น `toat-sandbox-db` จากนั้นคลิกสร้าง
4. เมื่อสร้างเสร็จสิ้น ท่านจะได้รับข้อมูลสำคัญ 2 อย่าง:
   * **Database URL**: ลิงก์เชื่อมต่อ (เช่น `libsql://toat-sandbox-db-yourusername.turso.io`)
   * **Auth Token**: รหัสผ่านเชื่อมต่อฐานข้อมูล (กดปุ่ม Generate Token เพื่อสร้างขึ้นมาใหม่)

### ขั้นตอนที่ 2: อัปโหลดโค้ดขึ้น GitHub
1. สร้างคลังข้อมูลใหม่ (New Repository) บน GitHub ของท่าน ตั้งชื่อตามต้องการ (เช่น `toat-sandbox-tracker`)
2. ส่งโค้ดในโฟลเดอร์นี้ขึ้นไปยัง GitHub Repository ที่สร้างขึ้น
   ```bash
   git init
   git add .
   git commit -m "Migrate to Turso Database"
   git branch -M main
   git remote add origin <URL_REPOSITORY_ของท่าน>
   git push -u origin main
   ```

### ขั้นตอนที่ 3: สั่ง Deploy บน Render.com
1. ล็อกอินเข้าใช้งานหน้าเว็บ [Render.com](https://dashboard.render.com)
2. คลิกปุ่ม **New +** ที่มุมขวาบน -> เลือก **Blueprint**
3. เลือกคลังเก็บข้อมูล (Repository) ของท่านบน GitHub
4. ระบบของ Render จะอ่านไฟล์ `render.yaml` อัตโนมัติ และแสดงรายการตั้งค่าขึ้นมา โดยมีช่องให้กรอก Environment Variables ดังนี้:
   * **TURSO_DATABASE_URL**: กรอก *Database URL* ที่ได้จากขั้นตอนที่ 1
   * **TURSO_AUTH_TOKEN**: กรอก *Auth Token* ที่ได้จากขั้นตอนที่ 1
5. คลิกปุ่ม **Apply** ด้านล่าง
6. ระบบจะเริ่มดำเนินการ Build และ Deploy โครงการขึ้นเซิร์ฟเวอร์จริงทันที (ใช้เวลาประมาณ 2-3 นาที) ลิงก์เข้าใช้ระบบของท่านจะแสดงอยู่ด้านบนสุดของหน้าเว็บครับ!
