# ระบบติดตามและบริหารโครงการ TOAT Sandbox

ระบบเว็บแอปพลิเคชัน Full-stack สำหรับติดตามความคืบหน้าโครงการ งบประมาณจัดสรร แผนงาน Gantt Chart การรายงานผลรายเดือน และระบบการตรวจสอบบทบาทผู้รับผิดชอบโครงการ (RBAC) พร้อมระบบแจ้งเตือนเมื่อเลยกำหนดส่งรายงานสำหรับรอบเดือน

---

## 1. คุณสมบัติของระบบ (Key Features)

1. **Authentication & Authorization (RBAC)**:
   - **Admin**: สามารถเข้าถึงการปรับปรุงบทบาท บัญชีผู้ใช้ การตั้งค่ากำหนดวันส่งรายงาน และมอบหมายโครงการให้พนักงาน
   - **Executive (ผู้บริหาร)**: มองเห็นแดชบอร์ดสรุปรายจ่ายโครงการ ความคืบหน้าภาพรวมทั้งหมด แสดงความคิดเห็นและบันทึกข้อเสนอแนะอย่างเป็นทางการใต้รายงาน และอนุมัติรายงานฉบับเดือน (Approved/Reject)
   - **Project Submitter (Staff)**: จัดทำโครงการ สมาชิกผู้เกี่ยวข้อง บันทึกข้อมูลแผน Gantt Chart อัปเดต % ความคืบหน้า จัดส่งรายงานประจำเดือน
2. **Project Workspace & Visual Gantt Chart**:
   - หน้าติดตามการดำเนินงานแบ่งออกเป็น แท็บข้อมูลสมาชิกแท้จริง, งบประมาณคงเหลือคำนวณอัตโนมัติ (Rollup), และแผนภูมิ Gantt Chart ที่คำนวณช่วงเวลาวาดกราฟเฉลี่ยอัตโนมัติ
3. **A4 Printable Monthly Report**:
   - หน้ารายงานแบบทางการจัดองค์ประกอบตามมาตรฐานกระดาษ A4 แนวตั้งสำหรับสั่งพิมพ์ออก PDF ได้ผ่านฟังก์ชันเบราว์เซอร์
4. **Cron Deadline System Notifications**:
   - ระบบเช็กค้างส่งรายงานเบื้องหลังของฝั่งเซิร์ฟเวอร์ พร้อมแผงควบคุม Admin ในการคลิกปุ่มเพื่อทดสอบส่งแจ้งเตือนและจำลองส่งอีเมลแจ้งเตือนได้ทันที

## 2. โครงสร้างโปรเจกต์ (Project Structure)

## 3. ขั้นตอนการติดตั้งและเปิดใช้งานระบบ (Installation & Setup)

1. **โหลด NVM และเปลี่ยน Node.js (หากใช้หลายรุ่น)**:
   ```bash
   export NVM_DIR="$HOME/.nvm"
   [ -s "$NVM_DIR/nvm.sh" ] && \. "$NVM_DIR/nvm.sh"
   nvm use 20
   ```

2. **ติดตั้ง Dependencies**:
   ```bash
   npm install
   ```

3. **ตรวจสอบความสมบูรณ์ของฐานข้อมูล**:
   ```bash
   node test_db.js
   ```
   *(จะพิมพ์ข้อความ PASS ครบทุกข้อหากเสร็จสมบูรณ์)*

4. **เริ่มการทำงานของเซิร์ฟเวอร์**:
   ```bash
   npm start
   ```
   ระบบจะทำงานผ่านพอร์ต `http://localhost:3000`

## 5. เทคนิคและรายละเอียดโครงสร้างสไตล์ที่นำมาใช้

- ** dialog closedby **: เพื่อรองรับการคลิกนอกกรอบเพื่อซ่อน modal หน้าต่างแบบ native โดยระบบได้ติดตั้งระบบ fallback event-coordination รองรับบราว์เซอร์ Safari (iOS) แล้วโดยอัตโนมัติ
- **@media print**: มีการคำนวณตัดขอบและซ่อนเมนูนำทางเมื่อเข้าสู่โหมดพิมพ์ของบราว์เซอร์ เพื่อความพร้อมในการ Export รายงาน PDF ขนาด A4
